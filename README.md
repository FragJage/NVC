Nicky Voice Control
===================
Nicky Voice Control (NVC) est un assistant vocal qui permet de piloter votre PC, votre domotique et plus encore. 

Introduction
============
L'objectif de NVC est d'être capable de fonctionner sur des petites plateforme telle que le Raspberry PI.
Pour obtenir des bons temps de réponses, la reconnaissance vocale et le parser d'intention sont locals, toutefois les prochaines versions proposeront des solutions plus puissantes mais via internet.
Le système de plugin permet d'augmenter le possibilité de NVC.

Caractéristiques
================
Nom du bot personnalisable
La reconnaissance vocale utilise Sphinx
La synthèse vocale utilise PicoVoice
Grammaire et dictionnaire pour Sphinx générer dynamiquement
Code C++ portable

Portabilité
===========
Actuellement compilé sous Windows avec Mingw en évitant les dépendances de l'OS.
Devrait compiler sous linux et Raspberry PI.
